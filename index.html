First-time setup

This app needs a free Supabase database so everyone sees the same data. Follow the steps below — it takes about 5 minutes.

Setup steps: 1. Go to supabase.com → sign up free
2. Click New project, give it any name, set a database password
3. Wait ~1 min for it to provision, then go to Project Settings → API
4. Copy your Project URL and anon public key into the fields below
5. Go to the SQL Editor in your Supabase project and run this query:
CREATE TABLE guild_state (
  id TEXT PRIMARY KEY DEFAULT 'main',
  data JSONB NOT NULL
);

INSERT INTO guild_state (id, data) VALUES ('main', '{
  "team": [],
  "remaining": [],
  "done": [],
  "current": null,
  "adminPw": "captain123"
}'::jsonb);

ALTER TABLE guild_state ENABLE ROW LEVEL SECURITY;

CREATE POLICY "public read"
  ON guild_state FOR SELECT USING (true);

CREATE POLICY "public write"
  ON guild_state FOR UPDATE USING (true);
6. Paste your URL and key below and click Connect
